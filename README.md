### Описание датасета
#### Основные характеристики
- Размер: 12,330 строк × 18 столбцов
- Целевая переменная: Revenue (True/False - совершена ли покупка)
- Тип задачи: Бинарная классификация
- Баланс классов: Несбалансированный (~15% покупок)

#### Атрибуты датасета
##### 1. Поведенческие метрики
|Атрибут |	Тип |	Описание|
|--------|------|-----------|
|Administrative |	int |	Количество административных страниц |
|Administrative_Duration |	float |	Время на адм. страницах (сек) |
|Informational |	int |	Количество информационных страниц |
|Informational_Duration |	float |	Время на инф. страницах (сек) |
|ProductRelated |	int |	Количество страниц продуктов |
|ProductRelated_Duration |	float |	Время на страницах продуктов (сек) |
##### 2. Метрики Google Analytics
|Атрибут | 	Тип	  | Описание |
|--------|--------|----------|
|BounceRates | 	float |	Показатель отказов (%) |
|ExitRates | float  |	Показатель выходов (%) |
|PageValues |	float |	Ценность страницы |
##### 3. Временные и контекстные атрибуты

##### 4. Технические атрибуты

##### 5. Демографические атрибуты

### Цели проекта
#### 1. Основные цели
- Разведочный анализ данных (EDA)
- Анализ распределений признаков
- Исследование корреляций
- Выявление паттернов в поведении пользователей
#### 2. Предобработка данных
- Обработка пропущенных значений
- Кодирование категориальных признаков
- Нормализация/стандартизация данных
#### 3. Построение ML-моделей
- Бинарная классификация
- Сравнение разных алгоритмов
- Оценка качества моделей
#### 4. Разработка пайплайна
- Создание воспроизводимого кода
- Документирование процесса
- Контейнеризация решения

### Структура исследования
1. notebooks/01_eda.ipynb - Разведочный анализ
2. notebooks/02_preprocessing.ipynb - Предобработка данных
3. notebooks/03_modeling.ipynb - Построение моделей
